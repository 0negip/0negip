## About Me

I am currently studying **Artificial Intelligence and Data Engineering** at the University of Pisa, after completing a bachelor's degree in **Applied Informatics and Data Analytics** at the University of Cagliari.

My academic and technical interests focus on artificial intelligence, machine learning, data engineering, explainable AI, knowledge graphs, recommender systems, distributed data processing, and performance-aware software systems.

I am particularly interested in the design of transparent, reliable, and data-driven systems, with attention to both model performance and interpretability. My work combines machine learning experimentation, data analysis, software engineering, and large-scale data processing.

## Selected Projects

### FoodNexus

Software project related to food knowledge graphs, connecting user-recipe interaction data with structured food product information.  
The project is aligned with my interests in knowledge graphs, recommender systems, data integration, and interpretable data-driven applications.

### CookMatch

Academic project for the Large Scale and Multi-Structured Databases course at the University of Pisa.  
The project reflects my current work in data engineering, structured and semi-structured data management, and database-oriented application development.

### XAI DeepFake Classifier

Machine learning and explainable AI project focused on the analysis and interpretation of deepfake classification models.  
The work explores how explanation methods can be used to inspect model behavior and evaluate the consistency of visual explanations.

### Frogger in Parallel

Classic Frogger reimplemented in C with `ncurses`, using both multithreaded and multiprocess architectures.  
The project focuses on low-level concurrency, process coordination, synchronization, and parallel software design.

### Shared-Bus Parallel Computing Simulation with OMNeT++

Performance evaluation project focused on the simulation and analysis of a shared-bus parallel computing architecture.  
The project is related to system modeling, performance analysis, and simulation-based evaluation.
